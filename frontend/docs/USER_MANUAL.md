## 1. 시스템 개요

### 1.1 시스템 소개
e1soft LLM은 제조 품질 불량 분석을 위한 AI 기반 분석 시스템입니다. 고객 불량 접수부터 데이터 수집, 분석, 보고서 작성까지 전 과정을 지원합니다.

### 1.2 주요 기능
- **고객 불량 접수**: 이메일을 통한 불량 접수 및 등록
- **데이터 수집**: ERP, MES, 품질 시스템에서 데이터 자동 수집
- **AI 기반 분석**: LLM을 활용한 불량 원인 분석 및 개선안 제시
- **협업 지원**: 담당자 확인, 메일 발송, 결재 상신 기능

### 1.3 시스템 요구사항
- **브라우저**: Chrome, Firefox, Edge, Safari (최신 버전)
- **화면 해상도**: 1280x720 이상 권장
- **네트워크**: 인터넷 연결 필요

## 2. 시작하기

### 2.1 로그인

1. 웹 브라우저에서 `http://localhost:3000` 접속
2. 이메일과 비밀번호 입력
3. **로그인** 버튼 클릭

```
┌─────────────────────────────────┐
│         e1soft LLM 로그인        │
├─────────────────────────────────┤
│  이메일: [________________]     │
│  비밀번호: [________________]    │
│                                 │
│        [ 로그인 ]                │
│                                 │
│  계정이 없으신가요? 회원가입      │
└─────────────────────────────────┘
```

### 2.2 회원가입

1. 로그인 화면에서 **회원가입** 링크 클릭
2. 이름, 이메일, 비밀번호 입력
3. **회원가입** 버튼 클릭

> **참고**: 회원가입 후 관리자 승인이 필요할 수 있습니다.

### 2.3 로그아웃

1. 화면 우측 상단의 프로필 아이콘 클릭
2. **로그아웃** 버튼 클릭

## 3. 메인 화면 구성

### 3.1 전체 레이아웃

```
┌─────────────────────────────────────────────────────────────┐
│ [≡] e1soft LLM                           [모델] [⚙] [👤]   │  ← 헤더
├──────────┬─────────────────────────────────┬────────────────┤
│          │                                 │                │
│  채팅    │      메인 콘텐츠 영역            │   모델         │
│  이력    │                                 │   선택         │
│          │                                 │                │
│          │                                 │                │
├──────────┴─────────────────────────────────┴────────────────┤
│                        하단 영역                            │
└─────────────────────────────────────────────────────────────┘
```

### 3.2 헤더 영역

| 요소 | 설명 |
|------|------|
| **[≡]** | 좌측 사이드바 토글 (채팅 이력) |
| **e1soft LLM** | 로고 클릭 시 메인 화면으로 이동 |
| **[모델]** | 우측 사이드바 토글 (모델 선택) |
| **[⚙]** | 관리자 설정 (관리자만 표시) |
| **[👤]** | 사용자 프로필 및 로그아웃 |

### 3.3 좌측 사이드바 (채팅 이력)

- 이전 대화 목록 표시
- 새 대화 시작 버튼
- 대화 검색 기능
- 대화 그룹화 기능

### 3.4 우측 사이드바 (모델 선택)

- 사용 가능한 LLM 모델 목록
- 모델 선택 및 변경
- 모델별 상세 정보

## 4. 품질 분석 워크플로우

### 4.1 분석 워크플로우 카드

메인 화면에서 6개의 분석 단계 카드가 표시됩니다:

```
┌─────────────────┬─────────────────┬─────────────────┐
│  📧 고객불량    │  📊 데이터      │  🔍 기본        │
│     확인        │     수집        │     분석        │
├─────────────────┼─────────────────┼─────────────────┤
│  🔬 상세        │  📝 보고서      │  📁 과거        │
│     분석        │     작성        │     이력확인    │
└─────────────────┴─────────────────┴─────────────────┘
```

### 4.2 각 단계별 설명

#### 1단계: 고객불량 확인
- 고객사로부터 접수된 불량 메일 확인
- 불량 정보 등록 및 분석 케이스 생성
- 담당자 배정

#### 2단계: 데이터 수집
- ERP 출하 정보 조회
- MES 생산 실적 조회
- LOT 트래킹 정보
- 품질 검사 이력
- 불량 이력 조회
- 공정/설비 이력
- 개발/자재 이력

#### 3단계: 기본 분석
- AI 기반 1차 분석
- 고객 담당자 확인
- 품질 담당자 지정
- 귀책부서 확인
- 협의 메일 발송
- 분석결과 상신

#### 4단계: 상세 분석
- 현장 데이터 분석
- 집계 데이터 분석
- 심층 원인 분석

#### 5단계: 보고서 작성
- 과거 보고서 수집
- 집계 분석 데이터 수집
- 분석 보고서 작성
- 내부 결재
- 최종 확정

#### 6단계: 과거 이력확인
- 유사 불량 이력 조회
- 과거 데이터 연계
- 관련 보고서 연계

### 4.3 분석 대상 선택

1. 메인 화면 상단의 **분석 대상 선택** 드롭다운 클릭
2. 등록된 분석 대상 목록에서 선택
3. 선택한 대상의 정보 확인:
   - 고객사
   - 제품 모델
   - LOT ID
   - 불량 유형

```
┌────────────────────────────────────────┐
│ 분석 대상 선택                    [▼]  │
├────────────────────────────────────────┤
│ ● REG-2024-001 | Apple | DEAD_PIXEL    │
│ ○ REG-2024-002 | Samsung | MURA        │
│ ○ REG-2024-003 | LG | LINE_DEFECT      │
└────────────────────────────────────────┘
```

## 5. 데이터 수집 기능

### 5.1 데이터 수집 시작

1. **데이터 수집** 카드 클릭 또는 드롭다운에서 **데이터 수집** 선택
2. 분석 대상 정보 확인
3. **데이터 수집 시작** 버튼 클릭

### 5.2 수집 단계 선택

데이터 수집 시작 전, 수집할 단계를 선택할 수 있습니다:

```
수집 단계                    체크 해제 시 해당 단계 스킵
────────────────────────────────────────────────────
[✓] 1. ERP 출하 정보        출하 LOT 정보 조회
[✓] 2. MES 생산 실적        생산 실적 및 수율 정보 조회
[✓] 3. LOT 트래킹           LOT별 공정 흐름 추적
[✓] 4. 품질 검사            품질 검사 이력 조회
[✓] 5. 불량 이력            관련 불량 이력 조회
[✓] 6. 공정/설비 이력       공정 파라미터 및 설비 이력 조회
[✓] 7. 개발/자재 이력       설계 변경 및 자재 정보 조회
```

### 5.3 LOT 변경

수집 시작 전 다른 LOT으로 변경하거나 복수 LOT을 추가할 수 있습니다:

1. **LOT 변경** 버튼 클릭
2. 드롭다운에서 LOT 선택 또는 직접 입력
3. **추가** 버튼으로 복수 LOT 추가
4. 첫 번째 LOT이 주(主) LOT으로 사용됨

### 5.4 단계별 데이터 확인

각 수집 단계에서:

1. **데이터 조회 중...** 로딩 표시
2. 조회 결과 테이블 형태로 표시
3. **코멘트 추가** (선택사항) 입력 가능
4. 액션 버튼:
   - **처음으로**: 첫 화면으로 이동
   - **스킵**: 현재 단계 건너뛰기
   - **재조회**: 데이터 다시 조회
   - **확인 및 다음**: 다음 단계로 이동

### 5.5 최종 확인

모든 단계 완료 후:

1. 수집된 데이터 요약 확인
2. **데이터 재수집**: 다른 조건으로 재수집
3. **기본 분석 시작**: 수집 데이터 기반 분석 진행

## 6. 기본 분석 기능

### 6.1 기본 분석 화면 구성

```
┌─────────────────────────────────────────────────────────────┐
│ ◀ │ 기본분석 [진행중] │    Apple | OLED_67_FHD | DEAD_PIXEL │
├─────────────────────────────────┬───────────────────────────┤
│                                 │   기본분석 진행            │
│   1차분석 결과                  │   ──────────────────      │
│                                 │   진행률 2/6 완료          │
│   [채팅 메시지 영역]            │   ======================== │
│                                 │                           │
│                                 │   ☑ 1. 데이터수집 확인    │
│                                 │   ○ 2. 1차분석 결과       │
│                                 │   ○ 3. 고객담당자 확인    │
│                                 │   ○ 4. 품질/귀책부서      │
│                                 │   ○ 5. 협의 메일발송      │
│  ┌──────────────────────────┐  │   ○ 6. 분석결과 상신      │
│  │ 메시지 입력...      [전송] │  │   ──────────────────      │
│  └──────────────────────────┘  │   [ 다음 단계로 → ]        │
└─────────────────────────────────┴───────────────────────────┘
```

### 6.2 분석 단계별 기능

#### 1. 데이터수집 내용 확인
- 수집된 생산/품질 데이터 확인
- 좌측 사이드바에서 상세 데이터 조회

#### 2. 1차분석 결과 확인
- AI가 수집 데이터를 분석하여 결과 제공
- 불량 원인 추정
- 개선 방안 제시

#### 3. 고객담당자 확인
클릭 시 다음 정보 표시:
- 담당자 이름, 부서, 직책
- 이메일, 전화번호
- 선호 연락 방식
- 최근 커뮤니케이션 이력

#### 4. 품질담당자/귀책부서 확인
클릭 시 다음 정보 표시:
- 추천 품질담당자 정보
- 전문분야, 현재 담당 건수
- 추정 귀책부서
- 에스컬레이션 연락처

#### 5. 분석 협의 메일발송
클릭 시 다음 정보 표시:
- 메일 템플릿 선택
- 수신자 목록 (TO/CC)
- 메일 본문 미리보기
- 첨부파일 선택

#### 6. 분석결과 상신
클릭 시 다음 정보 표시:
- 결재선 (선임 → 수석 → 팀장)
- 보고서 구성 목록
- 보고서 미리보기
- 첨부 문서 목록

### 6.3 진행률 추적

- 각 단계 완료 시 체크 표시
- 상단 진행률 바 자동 업데이트
- 모든 단계 완료 시 **분석 완료** 버튼 활성화

## 7. 채팅 기능

### 7.1 메시지 입력

1. 하단 입력창에 질문 또는 요청 입력
2. **Enter** 키 또는 **전송** 버튼 클릭
3. AI 응답 대기 (로딩 애니메이션 표시)

### 7.2 마크다운 지원

AI 응답은 마크다운 형식을 지원합니다:
- **제목**: # 제목, ## 소제목
- **표**: 마크다운 테이블 형식
- **목록**: 번호 목록, 글머리 기호
- **코드**: 코드 블록 하이라이팅
- **강조**: **굵게**, *기울임*

### 7.3 표 데이터 엑셀 다운로드

테이블 데이터가 포함된 응답에서:

1. 테이블 위의 **📥 Excel 다운로드** 버튼 클릭
2. Excel 파일(.xlsx)로 자동 다운로드

### 7.4 대화 이력

- 좌측 사이드바에서 이전 대화 목록 확인
- 대화 클릭 시 해당 대화로 이동
- 대화 제목 자동 생성 (첫 메시지 기반)

## 8. 자주 묻는 질문

### Q1. 로그인이 되지 않습니다.
**A**: 이메일과 비밀번호를 정확히 입력했는지 확인하세요. 계정이 없다면 회원가입을 진행하거나 관리자에게 문의하세요.

### Q2. 데이터 수집 시 "데이터 없음"이 표시됩니다.
**A**: 해당 LOT에 대한 데이터가 시스템에 등록되어 있지 않을 수 있습니다. LOT ID를 확인하거나 관리자에게 문의하세요.

### Q3. AI 응답이 느립니다.
**A**: 네트워크 상태를 확인하세요. 복잡한 분석 요청의 경우 응답 시간이 길어질 수 있습니다.

### Q4. 특정 기능이 보이지 않습니다.
**A**: 사용자 권한에 따라 접근 가능한 기능이 다릅니다. 관리자에게 권한 확인을 요청하세요.

### Q5. 메일 발송이 되지 않습니다.
**A**: 현재 버전에서는 메일 발송이 시뮬레이션으로 동작합니다. 실제 메일 발송을 위해서는 시스템 설정이 필요합니다.
