## 1. 시나리오 개요


### 1.2 주요 액터(Actor) 정의

| 액터 | 역할 | 주요 사용 기능 |
|------|------|---------------|
| 품질분석가 | 불량 접수 및 분석 수행 | 데이터 수집, 기본분석, LLM 채팅 |
| 품질담당자 | 분석 검토 및 귀책 결정 | 분석 결과 검토, 귀책부서 지정 |
| 시스템 관리자 | 시스템 설정 및 관리 | 사용자 관리, 모델 설정, RAG 구성 |
| 고객 담당자 | 고객사 커뮤니케이션 | 메일 발송, 회신 처리 |
